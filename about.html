About {{ app_name }}

{{ app_name }} is a course-based academic interaction system designed to improve classroom engagement with a lightweight, structured workflow: students submit comments/questions during a lecture, and lecturers view and resolve them.

Personas

  • Student (Learner): submit comments in real time, get quick feedback, optionally anonymous.
  • Lecturer (Admin/Content Manager): view, manage, and resolve questions to structure feedback.

Implemented user stories

  • US1: Submit comment (Student)
  • US2: View confirmation (Student)
  • US3: View comments list (Lecturer)
  • US4: Resolve comment (Lecturer)

Note: this is a prototype, meant for coursework demonstration rather than production deployment.
